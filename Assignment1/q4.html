1. html - The root element of an HTML document.
  2. head - Contains metadata about the HTML document.
  3. title - Sets the title of the HTML document.
  4. meta - Provides metadata about the document.
  5. link - Used to link external resources.
  6. script - Defines client-side JavaScript code.
  7. style - Contains inline CSS styles or references an external CSS file.
  8. body - Contains the main content of the HTML document.
  9. h1 to h6 - Headings of various levels.
  10. p - Represents a paragraph of text.
  11. a - Creates hyperlinks to other web pages or resources.
  12. img - Embeds images in the HTML document.
  13. ul - Defines an unordered list.
  14. ol - Defines an ordered (numbered) list.
  15. li - Represents a list item within ul or ol lists.
  16. table - Defines an HTML table to organize data.
  17. tr - Represents a table row within a table.
  18. td - Represents a table cell within a tr.
  19. form - Used to create interactive forms for user input.
  20. div - A generic container for grouping and styling HTML elements.
